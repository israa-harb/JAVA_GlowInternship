Integer Types:
1.Byte: this data type can store whole numbers from -128 to 127. This can be used instead of int or other integer types to save memory when you are certain that the value will be within -128 and 127, for example:
byte myNum = 100;
System.out.println(myNum);

2.Short: can store whole numbers from -32768 to 32767, for example:
short myNum = 5000;
System.out.println(myNum);

3.Int: can store whole numbers from -2147483648 to 2147483647. In general, and in our tutorial, the int data type is the preferred data type when we create variables with a numeric value, for example
int myNum = 100000;
System.out.println(myNum);
